# Bingo Game in C 🎲

A console-based Bingo game developed in C that allows two players to create custom bingo cards and play using randomly generated numbers.

## 🚀 Features
- Two-player interactive gameplay
- Custom bingo card input (3x3 grid)
- Input validation (no duplicates, range check)
- Random number generation (1–50)
- Dynamic card updates after each round
- Winner detection based on matched numbers

## 🧠 How It Works
1. Players enter their names and create a 3x3 bingo card
2. System generates random numbers in multiple rounds
3. Matching numbers are marked (converted to 0)
4. Final score is calculated based on matches
5. Player with highest matches wins

## 🛠 Technologies Used
- C Programming
- Standard Libraries:
  - stdio.h
  - stdlib.h
  - time.h
